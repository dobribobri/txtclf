Microsoft выпустила новый патч для Windows 10.
Американская компания Microsoft в экстренном режиме выпустила обновление системы безопасности для ОС Windows 10. Новый патч для системы поможет устранить уязвимость сетевого протокола, предназначенного для удаленного доступа к принтерам, файлам и другим ресурсам.
